# SecureLens MVP v1

## Features

### Technology Detection
- Detect framework
- Detect server

### SSL Analysis
- Certificate validation
- TLS version
- Expiry detection

### Security Header Analysis
- CSP
- HSTS
- X-Frame-Options
- Referrer-Policy

### Security Score
- Score calculation
- Risk levels

### AI Summary
- Risk explanation
- Recommendations

## User Flow

User submits URL
↓
Technology Detection
↓
SSL Analysis
↓
Header Analysis
↓
Score Calculation
↓
AI Summary
↓
Report
